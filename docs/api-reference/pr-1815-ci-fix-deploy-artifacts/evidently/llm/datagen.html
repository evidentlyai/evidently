class FewShotDatasetGenerator(evidently.llm.datagen.base.BaseLLMDatasetGenerator):

Dataset generator using few-shot learning approach.

Generates synthetic data by providing examples to an LLM and asking it to generate similar content. Useful for creating training or evaluation datasets when you have limited examples.
