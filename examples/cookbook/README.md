# Cookbook

This folder contains short, focused examples for specific Evidently functionality.

Use these notebooks to explore individual features, understand specific workflows, and quickly adapt examples to your own use case.

## Available examples

| Notebook | Description |
|----------|------------|
| [`metrics.ipynb`](./metrics.ipynb) | Basic examples of working with Evidently metrics |
| [`descriptors.ipynb`](./descriptors.ipynb) | Using descriptors for feature and data analysis |
| [`guardrails.ipynb`](./guardrails.ipynb) | Setting up guardrails for LLM applications |
| [`prompt_registry.ipynb`](./prompt_registry.ipynb) | Storing, versioning, and reusing prompts with the Prompt Registry |
| [`prompt_optimization_bookings_example.ipynb`](./prompt_optimization_bookings_example.ipynb) | Prompt optimization for a bookings use case |
| [`prompt_optimization_code_review_example.ipynb`](./prompt_optimization_code_review_example.ipynb) | Prompt optimization for code review tasks |
| [`prompt_optimization_tweet_generation_example.ipynb`](./prompt_optimization_tweet_generation_example.ipynb) | Prompt optimization for tweet generation |
| [`recsys_metrics.ipynb`](./recsys_metrics.ipynb) | Evaluation metrics for recommendation systems |
| [`regression_preset.ipynb`](./regression_preset.ipynb) | Using presets for regression tasks |
| [`datagen.ipynb`](./datagen.ipynb) | Generating synthetic data for experiments and testing |

## When to use the cookbook

Use the cookbook if you want to:

- try a specific Evidently feature in isolation,
- copy a minimal implementation pattern,
- understand a single evaluation or monitoring task without going through a full workflow.

For end-to-end examples, see the main [`examples`](../) folder and the [`tutorials`](../tutorials/) directory.
